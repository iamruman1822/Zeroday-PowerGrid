# Routes Folder Purpose

This folder contains Flask blueprints for API endpoints.

## File Purpose

- __init__.py: Marks routes as a Python package.
- dashboard.py: Combined dashboard endpoint with alerts and module summaries.
- forecast.py: Forecast timeseries and forecast summary endpoints.
- battery.py: Battery health, fleet metrics, cycles, and plot-serving endpoints.
- dispatch.py: Live dispatch prediction and sample explanation endpoints.
- explainability.py: SHAP/model image serving and available plot listing endpoints.
